⭐ ViaFabricPlus

ViaFabricPlus is a Minecraft mod that implements the ViaVersion projects into Fabric and provides tons of fixes to the existing protocol translation which can't be implemented in the original ViaVersion project.

These fixes consist of movement changes, block/entity collisions, rendering changes, and many more.

❮ ❯

For more details, visit the GitHub repository.
